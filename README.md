# DiceBot by Ch1zheo

DiceBot is a simple script for rolling a die in Telegram.

Installation:

1. Extract the files to a folder of your choice.

2. Install the required libraries:

    pip install -r requirements.txt

3. Copy `default.env.example` to `.env`.

4. Create a bot via @BotFather, follow the instructions, and obtain the token.

5. Paste the token into the `.env` file.

# Commands

/start - Greetings

/dice - Rolls the die and gives the result
